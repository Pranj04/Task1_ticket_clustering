# Ticket Clustering with NLP and Machine Learning

This folder contains the assignment-ready project structure. The main implementation lives in reusable modules under `src/`, with a CLI entry point at `main.py`.

Run from the workspace root:

```powershell
python main.py --input data/tickets.csv --clusters 10
```

Run from this folder:

```powershell
python main.py --input ..\data\tickets.csv --clusters 10
```

See the root `README.md` for the full overview, installation steps, module descriptions, and sample output.
